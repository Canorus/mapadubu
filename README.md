# 마파두부 mapadubu
[모두부](https://github.com/yuburoll/modubu) 기반의 38키 세미 스태거 스플릿 키보드

As owner of original design mentioned, this semi-staggered layout may is in need for Korean speakers mostly, due to popular input layout called Dubeolsik among Korean users. As a result, no English tutorial will be provided from here but you are free to use online translator to proceed. AI seems to work great for that these days.

- 38키
- 우측 ㅠ키
- 완전 무선
- [zmk 펌웨어](https://github.com/canorus/zmk-mapadubu)
