# 세 줄 요약

일단 양면 기판. 거버는 [여기](../kicad/mapadubu-gerber_v.2.2.zip)나 릴리즈에서 다운로드.

앞면에 스위치, 리셋 버튼
뒷면에 소켓, xiao ble, LED
다이오드, 슬라이드 스위치는 앞뒤 아무 데나 땜해도 상관없습니다.

# 유의사항

- 기판 두께는 1.2mm
- 다이오드 방향, LED GND 방향에 유의
- LED는 뒷면에 땜하지만 발광부가 윗면을 향하도록 해야 합니다.
- 케이스는 PLA를 권장 - ABS나 다른 재질의 경우 깨집디다.
- 핫스왑 소켓 방향에 유의 [참고 이미지](/images/hs_socket_direction.JPG)
- 하판은 최소 두께 요건 충족을 위해서 이상한 핀이 붙어있습니다. 적당히 분지르든 깎아내든 처리하시고 사용하시면 됩니다.

# xiao ble 납땜

1. xiao ble를 정위치에 놓고 테이프로 고정

![](/images/soldered_topview.JPG)

3. 테이프 제거하고 나머지 핀 납땜

# 케이스 조립

1. 인서트를 가열해서 삽입
2. PCB를 놓고 스탠드 오프 장착
3. 하판을 PCB에 장착(상판 쪽에서 접근, 아직 스위치 끼우지 마세요)
4. 하판을 스탠드오프에 장착
5. 스위치 장착
